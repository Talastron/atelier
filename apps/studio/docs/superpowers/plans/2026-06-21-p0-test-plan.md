## Part 0 — Unblock dev: register your App Check debug token

**Symptom you saw:** `Could not verify the app with Firebase. Try refreshing the page.` in the Daily Brief card. In DevTools: `App Check debug token: 14a19767-5818-4122-b301-70c1854f89da`.

**Why:** Vite dev mode auto-generates an App Check debug token (see `src/firebase.js:65`). That token has to be registered in Firebase Console once per browser/dev install, otherwise every Firestore + Gemini call returns 403.

**Fix (one-time, ~30 seconds):**
1. Open https://console.firebase.google.com → your Atelier project
2. Build → **App Check** → click your web app → **Manage debug tokens**
3. Click **Add debug token**, paste `14a19767-5818-4122-b301-70c1854f89da`, give it a name like "Sibylle's laptop — Edge"
4. Hard refresh `localhost:5173` (Ctrl+Shift+R) — no 403s in the console
5. The Daily Brief should now compose on first wardrobe load

**Note:** I also shipped a defensive change (`p0/daily-brief` head) that **silently swallows auto-compose errors** so the alarming "Could not verify…" banner no longer appears on every page load. The error still appears in the DevTools console (so you can see it when debugging) and on manual "Try again" / "Compose another" clicks. The user-facing UI just doesn't get cluttered by a config issue.

**If production also fails** (you mentioned both preview and production), check in Firebase Console:
- **App Check → Web Apps → reCAPTCHA v3**: is your prod domain (e.g. `edit.myatelier.style`) registered?
- The `VITE_RECAPTCHA_SITE_KEY` env var in your prod build must match the site key shown in App Check → Web Apps
- **AI Logic → API tab**: App Check enforcement should be **Enabled** (otherwise tokens are checked but not required — which is fine in dev but should be on in prod)
- Hosting → your domain must be in **Authentication → Settings → Authorized domains** AND in App Check's allowed origins for reCAPTCHA

If everything above is set and Gemini still 401s in prod, the most likely culprit is a stale build that doesn't have the right reCAPTCHA site key embedded — `npm run build && firebase deploy --only hosting` with the prod `.env.production` in place.
